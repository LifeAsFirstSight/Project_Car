# Project_Car
用于七牛云秋招嵌入式议题的代码管理

***day1***

大概了解一下内容要求结合现有模块进行一个购买

***day2***

写一个简单的demo 比如超身边传感器配合舵机经行的一个避障功能，和通过l298n对电机一个正反转向，考虑到按照路径走（类似于循迹），又加购了一个舵机控制的前轮转向，小车采用阿克曼结构。前轮只负责控制前进方向，后轮输出动力。研究一下esp32-cam识别路径的可行性，保守起见，目前只实现通过wifi推流到QT界面做展示。研究下rgb传感器的例程。

***day3***
